Installation Guide

After unzipping Template Files, you will find our product DreamsPos->ci4(Source code), Documentation in the following Main Files folder.

Once unzipped the files, run the command to install the composer packages: 

composer install (or) composer update

In the root directory, you will find a file named .env, fill your DB credentials 

database.default.hostname = localhost
 
database.default.database = DreamsPos-ci

database.default.username = root

database.default.password = 

database.default.DBDriver = MySQLi

Open the terminal in your root directory(DreamsPos) & and run the following command for migrate the table: 

php spark migrate

For running a project,run the command:

php spark serve
