DreamsPOS - POS & Inventory Management Admin Dashboard Template

A point of sale admin template can help organize and track inventory in a business. The template includes fields for items, quantities, prices, and descriptions. It also includes fields for customer information.

Installation Guide

  • Node.js and NPM :
    If NodeJS is not installed then download it from https://nodejs.org/. NPM comes bundled with Node.js
  • NodeJS :
    You can install NodeJS by executing from your command windowhttps://github.com/nodejs
NPM Packages
  • Before proceding you'll need to install npm packages. You can do this by running npm install from the root of your project to install all the necessary dependencies.
npm install
Development Server
  • Run npm start for a dev server. Navigate to localhost:3000/. The app will autometically reload if you change any of the source files.
npm start
